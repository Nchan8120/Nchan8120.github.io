GBC Smart Waste Sorting

Overview

  • This project is a garbage classification app built as part of our AASD 4013 Agile Methodologies course. It follows Scrum practices, with collaborative sprints, version control, model training, and evaluation pipelines.
  • The Trashnet Datset has 6 classes: cardboard, glass, metal, paper, plastic, and trash, with roughly 2,500 images organized in folders by class.
  • Trained a CNN to classify these images with >70% accuracy.
  • Used PySpark and Snowflake for large-scale querying and pattern discovery involving substance abuse, collision types, and time-based factors.

Technologies Used

  • Python, TensorFlow, Keras, Pandas
  • Jupyter Notebooks
  • Flask/FastAPI (backend)
  • Streamlit (frontend)
  • Jira (task tracking)
