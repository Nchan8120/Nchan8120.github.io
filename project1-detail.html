Smart Parking Space Detector

Overview

  • Developed an AI-powered web application to detect parking spaces and classify occupancy status using computer vision and deep learning.
  • Built a two-stage pipeline with YOLOv5 for parking space detection and a custom CNN for occupancy classification, achieving 97% end-to-end accuracy.
  • Designed a drag-and-drop Gradio interface that outputs annotated images and detailed parking statistics.
  • Ensured high performance with real-time analysis, robust to varied parking lot conditions, and optimized for usability and accuracy.

Technologies Used

  • YOLOv5
  • Custom Convolutional Neural Network (TensorFlow CNN)
  • Gradio (web UI)
  • OpenCV, NumPy, Pandas
  • Python
